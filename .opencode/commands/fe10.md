## Deliverables

### 1. Team Management — `frontend/app/portal/team/page.tsx`

**Current problem:** Invite form sits inline below the table — cramped and hard to
validate. The "Remove" button is red with no confirmation.

**Redesign:**

`<PageHeader title="Team Members" action={isAdmin && <Button variant="primary" size="sm">+ Invite Staff</Button>} />`

**Team table:** `<Table>` with columns:

| Column | Content |
|---|---|
| Name | Full name |
| Email | Email address (`text-text-400`) |
| Role | `<StatusBadge>` — "admin" → blue, "officer" → gray |
| Status | `<StatusBadge status={account_status}>` |
| Actions | Remove button (admin only) |

Remove: `<Button variant="ghost" size="sm"><Trash2 size={16} className="text-danger" /></Button>`
Opens confirmation modal: "Remove {name} from the team? They will lose access immediately."
API: `DELETE /api/v1/universities/{id}/staff/{staff_id}/`
On success: remove from list optimistically.

**+ Invite Staff modal** (opens when header button clicked):

```
<Modal title="Invite Staff Member">
  <FormField label="Full Name" required>
    <Input type="text" />
  </FormField>
  <FormField label="Email Address" required>
    <Input type="email" />
  </FormField>
  <FormField label="Role" required>
    <Select>
      <option value="officer">Admissions Officer</option>
      <option value="admin">University Admin</option>
    </Select>
  </FormField>
  [Cancel] [Send Invitation]
</Modal>
```

API: `POST /api/v1/universities/{id}/staff/` with `{ email, full_name, permission_level }`
On success: close modal + show `<Alert variant="success">Invitation sent to {email}.</Alert>` inline
On error: show `<Alert variant="danger">{error}</Alert>` inside the modal

**Empty state:** `<EmptyState icon={<Users>} heading="No team members" description="Invite staff to manage applications and programs." action={{ label: '+ Invite Staff', onClick: openModal }} />`

### 2. Global Error Boundary — `frontend/app/error.tsx`

Next.js error boundary for unexpected runtime errors.

```tsx
'use client'
export default function GlobalError({ error, reset }) {
  return (
    <PublicShell>
      <div className="flex flex-col items-center justify-center py-24 text-center">
        <AlertCircle size={48} className="text-danger mb-4" />
        <h1 className="text-2xl font-display font-bold">Something went wrong</h1>
        <p className="text-text-600 mt-2 max-w-md">
          An unexpected error occurred. Please try again or contact support if the problem persists.
        </p>
        <div className="flex gap-3 mt-6">
          <Button variant="primary" onClick={reset}>Try Again</Button>
          <Button variant="secondary" href="/">Go Home</Button>
        </div>
      </div>
    </PublicShell>
  )
}
```
